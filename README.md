# ML Attendance Certification System

## Overview

The ML Attendance Certification System automates student certification eligibility analysis using Machine Learning. The system processes attendance records, performs feature engineering, trains multiple machine learning models, evaluates model performance, and automatically generates certification reports.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn
* Joblib
* Jupyter Notebook

## Project Workflow

1. Data Preprocessing
2. Feature Engineering
3. Exploratory Data Analysis
4. Model Training
5. Model Evaluation
6. Certification Prediction
7. Report Generation

## Machine Learning Models

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier

## Features Used

* Total Sessions
* Sessions Attended
* Average Attendance Minutes
* Attendance Percentage

## Generated Outputs

* Certified Students List
* Uncertified Students List
* Final Certification Report
* Model Evaluation Report
* Confusion Matrix Visualization

## How to Run

```bash
python main.py
```
